/api/friends/:friend_guid/:friend2_guid

Method: GET

Returns with :friend_guid is friends with :friend2_guid. Replace :friend_guid and :friend2_guid with the guid of the authors you want to see if they are friends.
Expected Input
No input required
Expected Output
{  
   "query":"friends",
   "authors":[  
      "9de17f29c12e8f97bcbbd34cc908f1baba40658e",
      "8d919f29c12e8f97bcbbd34cc908f19ab9496989"
   ],
   "friends":"YES"
}

/api/friendrequest

Method: POST

Sends a friend request from the author to the friend specified.
Expected Input
{  
   "query":"friendrequest",
   "author":{  
      "id":"8d919f29c12e8f97bcbbd34cc908f19ab9496989",
      "host":"http://127.0.0.1:5454/",
      "displayname":"Greg"
   },
   "friend":{  
      "id":"9de17f29c12e8f97bcbbd34cc908f1baba40658e",
      "host":"http://127.0.0.1:5454/",
      "displayname":"Lara",
      "url":"http://127.0.0.1:5454/author/9de17f29c12e8f97bcbbd34cc908f1baba40658e"
   }
}
Expected Output
HTTP/1.1 200 OK
